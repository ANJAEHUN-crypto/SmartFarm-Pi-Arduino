# 7차 패치 상세

- **배포 시점**: 7차
- **요약**: 토양 센서를 그래프 없이 실시간값·7일 평균만 표시하도록 개편

---

## 1. 그래프 제거

- **index.html**
  - Chart.js CDN 스크립트 제거
  - `badge-chart-grid` 및 7개 `<canvas>`(chartTemp, chartHumi, chartEC, chartPH, chartN, chartP, chartK) 제거
- **app.js**
  - `MAX_POINTS`, `Y_AXIS`, `makeChart`, `ALL_CHARTS`, 기존 `refreshBadgeChart`(차트 갱신) 전부 삭제
- **style.css**
  - `.badge-chart-grid`, `.chart-cell`, `.chart-title` 제거

---

## 2. 실시간·평균 표시 UI

- **표시 방식**
  - 지표별 카드: **실시간** = API history의 마지막 1건, **7일 평균** = 동일 history 전체의 해당 필드 평균
  - 지표: 온도(°C), 습도(%), EC(µS/cm), pH, N, P, K (데이터 없으면 `-` 표시)
- **index.html**
  - `badge-section` 내부를 `badge-value-grid` + 7개 `badge-value-card`로 교체
  - 각 카드: 라벨 + "실시간" 값 요소(id: badgeXxxCurrent) + "7일 평균" 값 요소(id: badgeXxxAvg)
- **app.js**
  - `refreshBadgeValues()`: `/api/badge/history?days=7&limit=3000` 호출 후 마지막 건 파싱 → 실시간, 전체 평균 계산 → `setBadgeEl()`로 DOM 갱신
  - 5초마다 `refreshBadgeValues()` 호출
- **style.css**
  - `.badge-value-grid`, `.badge-value-card`, `.badge-value-label`, `.badge-value-row` 스타일 추가
